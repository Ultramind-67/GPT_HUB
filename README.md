MTS AI Smart Workspace — это интеллектуальная операционная среда, разработанная для трансформации корпоративных рабочих процессов МТС. Платформа представляет собой единый мультимодальный хаб, который автоматизирует рутину, обеспечивает безопасность данных и сохраняет долгосрочный контекст взаимодействия.

🌟 Концепция проекта

В отличие от стандартных интерфейсов для LLM, наш проект реализует концепцию "ИИ-Оркестрации". Пользователь взаимодействует с системой через естественный язык, а внутренний слой маршрутизации (MWS Smart Router) самостоятельно определяет, какую модель или автономного агента задействовать для решения задачи.

🤖 Экосистема автономных агентов

Платформа включает в себя 11 специализированных инструментов, интегрированных в единый User Flow:

🧩 Управление и Маршрутизация

MWS Smart Router (Routing Middleware): Интеллектуальный фильтр, который анализирует интент пользователя. Автоматически переключает контекст на Vision (VLM) при получении изображений или на ImageGen при запросах на визуализацию, сохраняя текстовую аналитику на базе gpt-oss-120b.

Persistent_Memory: Долгосрочная память на базе файловой системы (long_term_memory.txt). Обеспечивает сохранение личных данных, предпочтений и истории проектов даже после удаления чатов или перезагрузки системы.

🛠 Инженерные и ИТ-инструменты

📁 Project_Architect: Автономный DevOps-инженер. Проектирует структуру IT-проектов и физически создает директории и файлы на жестком диске сервера (в папку /output).

🧪 QA_Automation_Pro: Агент контроля качества. Анализирует программный код и автоматически генерирует Unit-тесты, обеспечивая стабильность разработки.

📊 Diagram_Generator: Визуализатор логики. Преобразует текстовые описания в профессиональные графические схемы Mermaid.js прямо в интерфейсе чата.

🔍 Аналитика и Исследования

🛰 Deep_Research: Аналитический модуль глубокого поиска. Агрегирует данные из 5+ независимых источников интернета, проводит сравнительный анализ и выдает синтезированный отчет.

🌐 Web_Scraper: Инструмент экстракции данных. Проводит глубокий парсинг URL-ссылок, очищая контент от мусора (реклама, скрипты) для чистого анализа моделью.

🎨 Presentation_Creator: Генератор интерактивных HTML-презентаций в стиле Artifacts. Позволяет просматривать слайды с навигацией прямо внутри окна чата через встроенный Preview.

🔒 Безопасность и Бизнес-логика

🛡 Security_Guardian: DLP-слой (Data Loss Prevention). Проактивно сканирует каждый запрос на наличие утечек API-ключей, паролей и персональных данных (ПДН) до отправки во внешние API.

🎙 Meeting_Secretary: Цифровой ассистент совещаний. Обрабатывает аудиофайлы (ASR Whisper), формирует протоколы встреч (MoM) и сохраняет список поручений в формате Markdown.

🚀 Инструкция по развертыванию (Clean Start)

Проект спроектирован для безопасного развертывания в любой среде. Конфигурация агентов и маршрутизатора поставляется в виде экспортированного JSON-слепка.

1. Требования

Установленный Docker и Docker Compose.

Свободные порты 8000 (UI), 8081 (Поиск) и 9099 (Pipelines).

2. Запуск контейнеров

Склонируйте репозиторий, перейдите в папку проекта и выполните сборку:

code
Bash
download
content_copy
expand_less
docker-compose up -d

Дождитесь загрузки (около 1-2 минут при первом старте).

3. Создание учетной записи

Откройте в браузере: http://localhost:8000

Нажмите Sign Up и создайте новый аккаунт (первый зарегистрированный пользователь автоматически получает права Администратора).

4. Инициализация Экосистемы Агентов (Импорт)

Чтобы активировать все 11 инструментов, брендинг МТС и умный роутер:

Войдите в систему.

В левом меню перейдите в раздел Рабочее пространство (Workspace) -> Модели (Models).

Нажмите кнопку Импорт (Import Model).

Выберите файл mts_ai_model_backup.json из корня репозитория.

Готово! В вашем чате появится модель MTS Smart Workspace, полностью настроенная и готовая к работе со всеми заявленными функциями.

🏗 Технологический стек

LLM Core: gpt-oss-120b, qwen2.5-vl, qwen-image, whisper-turbo-local.

Backend: Python 3.11, OpenWebUI Pipelines.

Search Engine: SearXNG (Self-hosted).

Data Layer: SQLite (Метаданные), Local FS (Persistent Storage).
