**MTS AI Smart Workspace** — это интеллектуальная операционная среда, разработанная для трансформации корпоративных рабочих процессов МТС. Это мультимодальный хаб, который автоматизирует рутину, обеспечивает абсолютную безопасность данных и сохраняет долгосрочный контекст взаимодействия.

---

## 🌟 Концепция: ИИ-Оркестрация
В отличие от стандартных чат-ботов, наша платформа реализует концепцию **AI Orchestration**. Внутренний слой маршрутизации **MWS Smart Router** анализирует запрос и автоматически делегирует его нужному агенту или модели (текст, код, зрение или поиск), обеспечивая бесшовный User Experience.

---

## 🤖 Экосистема автономных агентов
Платформа включает специализированные инструменты, интегрированные в единый рабочий поток:

### 🧩 Управление и Маршрутизация
*   **MWS Smart Router**: Интеллектуальный посредник. Переключает контекст на Vision (VLM) или ImageGen, сохраняя глубокую аналитику на базе `gpt-oss-120b`.
*   **Persistent Memory**: Долгосрочная память (`long_term_memory.txt`). Хранит предпочтения и историю проектов даже после перезагрузки системы.

### 🛠 Инженерные и ИТ-инструменты
*   **📁 Project Architect**: Автономный DevOps-инженер. Проектирует структуру и физически создает файлы проекта в директории `/output`.
*   **🧪 QA Automation Pro**: Анализирует код и автоматически генерирует Unit-тесты.
*   **📊 Diagram Generator**: Преобразует текст в профессиональные схемы `Mermaid.js` прямо в чате.

### 🔍 Аналитика и Исследования
*   **🛰 Deep Research**: Агрегирует данные из 5+ независимых источников интернета для синтезированных отчетов.
*   **🌐 Web Scraper**: Парсинг URL с очисткой контента от рекламы и лишнего кода.
*   **🎨 Presentation Creator**: Генерация интерактивных HTML-презентаций в стиле Artifacts с навигацией и Preview.

### 🔒 Безопасность и Бизнес-логика
*   **🛡 Security Guardian**: DLP-слой (Data Loss Prevention). Блокирует утечки API-ключей, паролей и ПДН до отправки во внешние сети.
*   **🎙 Meeting Secretary**: Обработка аудио через Whisper, формирование протоколов встреч (MoM) и списков поручений.

## 🚀 Инструкция по развертыванию

### 1️⃣ Системные требования
*   **Docker** & **Docker Compose**
*   **Свободные порты**: `8000` (UI), `8081` (Search), `9099` (Pipelines)

### 2️⃣ Запуск контейнеров
```bash
# Склонируйте репозиторий и запустите сборку
docker-compose up -d
```
*Ожидайте загрузки всех сервисов (1–2 минуты при первом старте).*

### 3️⃣ Создание учетной записи
1. Перейдите по адресу: [http://localhost:8000](http://localhost:8000)
2. Нажмите **Sign Up** и создайте аккаунт. 
   > *Примечание: Первый зарегистрированный пользователь автоматически становится Администратором.*

### 4️⃣ Инициализация агентов (Импорт)
Чтобы активировать 11 агентов и брендинг МТС:
1. Перейдите в **Workspace** -> **Models**.
2. Нажмите **Import Model**.
3. Выберите файл `mts_ai_model_backup.json` из корня репозитория.

*   **LLM Core**: `gpt-oss-120b`, `qwen2.5-vl`, `qwen-image`, `whisper-turbo-local`.
